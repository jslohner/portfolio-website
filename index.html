Async Redux

In this project I created a website that fetches and displays data from the Rick and Morty API. I used redux in order to manage state globally and also react-router. There are three pages corresponding to three endpoints from the Rick and Morty API, which are: "/character", "/episode", and "/location". The website is still very basic, but I built it in a way that would make it easy to add new features and to refine.
